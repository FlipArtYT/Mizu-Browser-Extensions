# 🧩 Mizu-Browser-Extensions
Source code of the Silk Mizu Browser Extensions

# 📺 Program List
- [x] `Sidenote` : Simple note taking application that supports multiple note management and note auto-save
- [x] `Quickcalc` : Basic sidebar calculator
- [ ] `SAPI Client` : Silk API client to chat with AI models
- [ ] `Timers` : Create and start timers in no time
- [ ] `Browserfetch` : Customizable fastfetch-like system information widget
- [ ] `Gemini API Client` : Easily accessible Gemini Client
- [ ] `WatchSensors` : Show live CPU and RAM usage in the bottom bar
- [ ] `Overcustomizable` : Customize almost every part of the browser's UI including theme, layout and additional widgets
